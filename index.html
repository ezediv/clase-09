git

  1. Entrar a la consola y pararnos en la carpeta que queremos trabajar
  2. si todavia no es un repostirio ejecutar git init. si ya es un respositorio, ir al paso 3
  3. trabajar en mis archivos normalmente, y cada tanto, ejecutar un git add . para agregar los archivos modificados al repostirio
  4. una vez que tengo una version copada, ejecuto git commit -m "mensaje"
  5. Volver al paso 3
